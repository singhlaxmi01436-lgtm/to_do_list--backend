# Ayushman’s To‑Do List Application ✨

A full‑stack to‑do list application built with **HTML, CSS, JavaScript, Node.js, Express, and MongoDB**.  
This project started as a simple static list and evolved into a polished application with modern UI, dark mode, progress tracking, and installable app features (PWA).

---

## 📖 Overview
This application helps users organize tasks into **Daily Goals** and **Long‑term Goals**, track progress visually, and enjoy a clean, recruiter‑friendly interface.  
It demonstrates my skills in **frontend design, backend API development, database integration, and deployment**.

---

## 🚀 Features
- **Task Management**: Add, mark complete, undo, and delete tasks
- **Categories**: Separate sections for Daily Goals and Long‑term Goals
- **Progress Bar**: Visual tracker of completed tasks
- **Dark Mode Toggle**: Switch themes for better UX
- **Modern UI**: Card layout, hover effects, icons, responsive design
- **PWA Ready**: Includes manifest + icons for installable app experience
- **MongoDB Integration**: Persistent storage of tasks

---

## 🛠 Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express
- **Database**: MongoDB (local or Atlas cloud)
- **Version Control**: Git & GitHub
- **Deployment Options**: GitHub Pages (frontend), Render/Vercel (backend), MongoDB Atlas (database)
